# Servidor HD dashboard v0.1

Panel personalizado basado en Swizzin dashboard.

## Instalación:

Está incluido en el pack de aplicaciones, para instalarlo:

```
sudo box install panel
```
### Dependencias

- Python 3.5+
- Flask
- Flask-htpasswd
- Flask-SocketIO
- eventlet
